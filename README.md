# Graitec-Advance-Design-2027.0
Graitec Advance Design — это программное обеспечение для проектирования и расчета стальных, бетонных и деревянных конструкций. Оно позволяет вам создавать трехмерные модели конструкций, выполнять статический и динамический анализ, генерировать документацию и отчеты по проекту. Оно также интегрируется с другими программами.
